Array陣列、Reference Data Type 比較、Array Methods

作業日期：2023/09/25

Array陣列

  • 將類似種類或用途的資料儲存在一起，並使用中括號識別 []

  • 在陣列裡面的值被稱為元素(element)

  • 屬於 refrence data type

    • Primitive Data Type and Reference Data Type(Object)

    • 處理變數方式：
      前者 by value（當宣告變數 b 並 assign 變數 a 給 b 。此時，b 的值是複製 a 過來。之後 b 任何更動都不會影響 a）；
      後者 by reference（當宣告變數 arr2 並將 arr1 賦值給 arr2，是把記憶體位置給 arr2，所以任何對 arr2 操作更動都會影響 arr1）

    • let arr1 = [ "a" , "b" , "c" ];
      let arr2 = arr1;
      arr2[0] = "d";
      console.log(arr1 , arr2); // ["d","b","c"],["d","b","c"]

Reference Data Type 比較

let arr1 = [ 1 , 2 , 3 ];
let arr2 = [ 1 , 2 , 3 ];
console.log(arr1 == arr2); // false 因為記憶體位置不同，即使值相同。若要比較必須使用迴圈比對裡面的值

Array Methods

.length 傳回陣列長度
.push(element) 將一個或多個元素加入到陣列最後一項，並回傳陣列的新長度
.pop() 不用放參數，會把陣列最後一項元素刪除，並回傳該元素
.shift() 刪除陣列第一項元素，並回傳刪除的元素
.unshift 加入一個或多個元素到陣列前頭，並回傳該陣列的新長度
