作業日期：2023/09/19

String Attributes(屬性) and Methods

  • .length：傳回字串長度

  • [n]：回傳索引值第n項，索引值從0開始(若string的長度為x，最後一個文字的index是x-1)

  • .slice(indexStart[,indexEnd])：提取字符串的一部分並將其作為新String返回，而不修改原始字符串。indexStart是inclusive，indexEnd是optional, exclusive

  • .indexOf(substring)：回傳 substring的索引值，若找不到則回傳 -1

  • .toUppercase()：回傳轉換成大寫的string(此方法不會影響string本身，可以利用reassignment變更)

  • .toLowercase()：回傳轉換成小寫的string(此方法不會影響string本身，可以利用reassignment變更)

  • .split(pattern)：將一個字符串變成一個有序的陣列，並回傳

  • .startsWith("")：確認字串是否以指定字串開頭，根據需要返回true or false

  • .endsWith("")：確認字串是否以指定字串結尾，根據需要返回true or false

  • .includes(str)：確認字串是否包含指定字串，根據需要返回true or false

  • .charCodeAt(n)：返回一個介於 0 和 65535 之間的整數，表示給定索引處n的 UTF-16 code unit。ASCII 維基百科
