Truthy and Falsy Values

Falsy values

  • false

  • 0,-0,0n(bigint)

  • "",'',``(empty string)

  • null

  • undefined

  • NaN

除此之外都是truthy values，包含[ ]empty array, { }empty object等等

Logical Operator與其他資料類型處理

&&

  • 如果左邊的值是true，則結果會回傳右側的值。如 console.log(5 && 100); // 結果：100

  • 如果左邊的值不是true，則結果會回傳左側的值。如 console.log(0 && 100); // 結果：0

||

  • 如果左邊的值能夠被轉換型態後是true，則結果會回傳左側的值。如 console.log(5 || 100); // 結果：5

  • 否則會回傳右側的值。（常見於用在後端處理資料是否有取得）如 console.log(data || "資料遺失或錯誤"); // 結果：資料遺失或錯誤

Coding Convention and Restrictions

Convention習慣

  • 變數與函數以小寫命名，若由兩個以上單字組成，則使用camelCase或underline

  • Operators周圍加上空白鍵。如：console.log(3 + 4);

  • 用分號結束一個statement

  • 常數constant全部大寫。如：const PI = 3.1415;

Restrictions限制

  • 不可由數字開頭

  • 不可包含hyphen(-)，已作為減法運算符

  • 不可使用保留字
