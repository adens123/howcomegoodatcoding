物件 Object

  • 每個 JavaScript 物件都具有屬性 (properties) 與方法 (methods) 可以操作
  • 物件的內容是由屬性組成的，而屬性是由 key-value pair 構成
  • value 可為任意資料型別的值，並且值是以參考（reference）的方式儲存


存取物件的屬性有兩種方式

                let Wilson = {
                  first_name : "Wilson",
                  last_name : "Ren",
                  age : 26,
                  is_married : true,
                  '!!12345!!': "Hello Hello",
                };

1.特性存取（property access）.

                console.log (Wilson .first_name);
                // Wilson

2.鍵值存取（key access）[ ]

                console.log (Wilson ["age"] ); 
                console.log (Wilson ["!!12345!!"]  ); 
                 
                // 26
                // "Hello World"

*若要使用一些包含特殊字符或動態產生的字串作為屬性名稱，就必須使用鍵值存取 [ ] 的方式

*屬性名稱只能是字串，若不是字串則會被強制轉為字串

*若存取物件中不存在的屬性，此時會回傳 undefined



this 用法

                let Wilson = {
                  first_name : "Wilson",
                  last_name : "Ren",
                  age : 26,
                  is_married : true,
                  '!!12345!!': "Hello Hello",

                  sayHi () {
                    console.log( this.first_name + "say hi" )
                  },
                  walk () {
                    console.log( this.first_name + "is walking..." )
                  }
                };

*當屬性的值改變時，用this就會跟著改變
