Array 陣列
  • 是 reference data type(copy by reference - assign後且變更後，原始值也會更著變更)
  • 不是 primitive data type(copy by value - assign後且變更後，原始值不變)
  • array的值稱為元素(element)
Reference Data Type 比較
let arr1 = [1, 2, 3];
let arr2 = [1, 2, 3];
console.log(arr1 == arr2); // false，雖然array內容一樣，但由於記憶體位置不同，所以會是false。如果真的需要比較裡面元素，則需進一步使用for loop的方式。
let arr1 = [1, 2, 3];
let arr2 = arr1; // assignment operation，因array是 copy by reference，所以此處會指向同一個記憶體位置。
console.log(arr1 == arr2); // true
Array Methods
  1. push(element) - 將一個或多個元素添加到陣列的末尾，並return陣列的新長度。
    let friends = ["Grace", "Mike", "Spencer", "Esther", "Slade"];
    friends.push("Jared");
    console.log(friends);

    (6) ['Grace', 'Mike', 'Spencer', 'Esther', 'Slade', 'Jared']
    0: "Grace"
    1: "Mike"
    2: "Spencer"
    3: "Esther"
    4: "Slade"
    5: "Jared"
    length: 6
  2. pop() - 從陣列中刪除最後一個元素並返回該元素。
    let friends = ["Grace", "Mike", "Spencer", "Esther", "Slade"];
    friends.pop();
    console.log(friends);

    (4) ['Grace', 'Mike', 'Spencer', 'Esther']
    0: "Grace"
    1: "Mike"
    2: "Spencer"
    3: "Esther"
    length: 4
  3. shift() - 從陣列中刪除第一個元素，並返回刪除的元素。
  4. unshift(element) - 將一個或多個元素添加到陣列的開頭，並return陣列的新長度。
