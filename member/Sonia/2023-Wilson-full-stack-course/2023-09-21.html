Truthy and Falsy Values

Falsy Values：

  1. false
  2. 0, -0, 0n (BitInt)
  3. "", '', `` (所有的empty string)
  4. null
  5. undefined
  6. NaN

除此之外的都是 truthy values，包含 [] empty array, {} empty object 等等。

let hasPremium = true;

if (hasPremium) {
   console.log("跳過廣告");
} else {
   console.log("加廣告");
}
Logical Operator 與其他資料類型處理
&&

// 如果左邊為true，則顯示右邊的值。

console.log( 3 && 10); // 10
console.log(-3 && -10); // -10

// 如果左邊為false，則顯示左邊的值。

console.log(0 && -10); // 0
||

// 如果左邊為true，則顯示左邊的值。

console.log(6 || 100); // 6

// 其他的情況，則回傳右邊的值。

console.log(NaN || 100); // 100
實用情況
let data1 = "Wilson";
console.log(data1 || "資料讀取失敗"); // Wilson

let data2 = "";
console.log(data2 || "資料讀取失敗"); // 資料讀取失敗
Coding Convention(慣例) and Restrictions(限制)
Convention慣例
  1. 變數與函數的名稱，全部小寫。若名稱由兩個以上的單字組成，則使用camelCase。(或也可使用 underline)。
  2. Operator周圍加上空白鍵。
  3. statement後面要加分號。
  4. Constants全部使用大寫。
  5. Class由大寫字母開頭。
Restriction(限制)
  1. 變數、函數名稱開頭不能數字。
  2. 變數、函數名稱不能有 hyphen。Hyphen已經預留給數字做剪髮運算。
  3. 變數、函數名稱不可使用reserved words。
