String 基本介紹
  1. String由字母或數字串接組成。
  2. String需使用單引號 或 雙引號。

    let name = "Sonia Wu"; // Sonia

    let name = 'Sonia Wu'; // Sonia

    let sentence = "he's a good guy"; // 當混用時，例如:句子中有單引號時，外面就用雙引號
  3. 兩個String之間的串接，使用 「+」。

    let first_name = "Sonia";
    let last_name = "Wu";
    // + is not addition, instead, + is string concatenation
    console.log(first_name + last_name); // SoniaWu
    console.log(first_name + " " + last_name); // Sonia Wu
  4. String之間的串接稱為 concatenation。簡稱 concat。
  5. String與String之間不能做「-」、「*」、「/」運算，若作此類運算，則會出現NaN，代表 Not a Number。(這是因為，JS還是會嘗試試算出某個數字結果，但兩個operand的值都不是數字，無法算出一個數字結果，所以會出現NaN)。
    [ 3 + 4 ； 3、4為operand；+ 為operator]

    let first_name = "Sonia";
    let last_name = "Wu";
    console.log(first_name * 5); // NaN

  6. String與Number之間，若是做「+」運算，則會變成String與String之間的concatenation。
    // 範例一
    let x = 10;
    let y = "10";
    // x data type is number, y data is String
    console.log(x + x); // 20
    console.log(y + y); // 1010
    console.log(x + y); // 1010
    console.log(y + x); // 1010

     // 範例二
    let n1 = 20;
    let n2 = 30;
    let name = "Johnson";
    let n3 = 10;
    let n4 = 15;

    console.log(n1 + n2 + name); // 50Johnson，程式碼由上到下、由左到右讀取
    console.log(n1 + n2 + name + n3 + n4); // 50Johnson1015
  7. \n可以換行。
    console.log("Sonia\nWu");
    Sonia
    Wu
Number Methods

JS是物件導向(OOP - Object-oriented programming)的程式語言，所以JS當中的數字可被視為是物件。數字常見可用的methods:

  1. toString() - returen 一個數字的 String。要記得加上()。
    let age = 27;
    console.log(age.toString()); // 27
    console.log(typeof age.toString()) // string
  2. toFixed(n) - return 被轉換後的數字，到小數點後第 n 位數。
    • return值為String。若希望return為數字，需要自己另外寫設定。
    • 注意! 0.1 + 0.2 === 0.3 為 false，因為二進制不能精確表示所有小數。 // 直接計算 0.1 + 0.2 結果會是 0.30000000000000004
    const pi = 3.1415926535;
    console.log(pi.toFixed(2)); // 小數點後第二位，結果為 3.14
    console.log(typeof pi.toFixed(2)); // string
String Attributes(屬性) and Methods

String有可用的attributes以及methods。常見的有下列11個：

// 其餘可至 MDN string 看。

  1. length - return String 的長度。
    let str1 = "SoniaWu";
    console.log(str1.length); // 7

    let str2 = "Sonia Wu";
    console.log(str2.length); // 8, 空白鍵也算

    let str3 = "";
    console.log(str3.length); // empty string，則 0
  2. [n] - return index 第 n 項的字。(index(索引)從0開始計算)。
    let str = "Sonia";
    S = [0]
    o = [1]
    n = [2]
    i = [3]
    a = [4]
    console.log(str[0]); // S

    console.log(str[-1]); // undefined
    console.log(str[10]); // undefined

    // 對於任意一個String，若string的長度為x，則最後一個文字的index會是 x-1。
    console.log(str[str.length - 1]); // a
  3. slice(indexStart [, indexEnd]) - 提取字符串的一部分並將其作為新String返回，而不修改原始字符串。indexStart是inclusive，indexEnd是optional, exclusive。
    let str = "Sonia";

    console.log(str.slice(3)); // ia, 以index從0開始算，從第3個開始取值

    console.log(str.slice(0,3)); // Son, 以index從0開始算，從0開始取值，取到3之前，不包含3
  4. indexOf(substring) - return substring的index位置。若找不到substring，則return -1。
    let str = "Sonia";

    console.log(str.indexOf("ia")); // 3

    console.log(str.indexOf("woo")); // -1
  5. toUpperCase() - return 轉換為大寫的 String。此方法不影響String本身。
  6. toLowerCase() - return 轉換為小寫的 String。此方法不影響String本身。
    let str = "Donaldson";

    console.log(str.toLowerCase()); // donaldson
    console.log(str.toUpperCase()); // DONALDSON

    str.toUpperCase();
    console.log(str); // Donaldson，此方法不影響String本身，所以仍是 Donaldson

    // 如要把str裡面的內容都要大寫，則用 reassignment：
    str = str.toLocaleUpperCase(); // reassignment
    console.log(str); // DONALDSON
  7. split(pattern) - 將一個字符串分成一個有序的array，然後return該array。
    let sentence = "Today is a good day";
    // "Today", "is", "a", "good", "day"

    let result = sentence.split(" ");// 每一個空白鍵，就切一刀
    console.log(result);
     (5) ['Today', 'is', 'a', 'good', 'day']
    0: "Today"
    1: "is"
    2: "a"
    3: "good"
    4: "day"
    length: 5
  8. startsWidth("s") - 確定String是否以指定字串s開頭，根據需要返回 true 或 false。
  9. endsWith("s") - 與startsWith()相同，但確認結尾。
    let sentence = "Today is a good day";

    console.log(sentence.startsWith("Today"));// true
    console.log(sentence.startsWith("today"));// false，大小寫有差別
  10. includes("str") - return true 如果String 內部包含str。
    let sentence = "Today is a good day";

    console.log(sentence.includes("good"));// true

  11. charCodeAt(n) - 返回一個介於 0 和 65535 之間的整數，表示給定索引處n的 UTF-16 code unit。 ASCII 維基百科
    let sentence = "Today is a good day";
    console.log(sentence.charCodeAt(0)); // 84, 查詢 "T" 的 UTF 16 Code
