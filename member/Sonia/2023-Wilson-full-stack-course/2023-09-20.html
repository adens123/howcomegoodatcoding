Boolean, Undefined, Null
  1. Boolean：true, false。
    let isPremium = true;
    console.log(isPremium);// true
    • binary 2, decimal 10, unary 1。
    • Unary operator [！] 可以將Boolean的值反轉。
      console.log(true); // true
      console.log(!true); // false
      console.log(!false); // true
    • JS 中的 unary operator typeof 可用來確認資料類型。
      console.log(typeof true); // boolean
  2. Undefined：
    • 已經宣告變數，卻沒有賦予initializer時，就會出現 undefined。
    • undefined 也是 JS 中的 functions 的預設 return value。
  3. null ： 用來代表某個故意不存在的值。
logical, comparison operator

常見運算符(operator)：

  1. assignment operator： =
  2. compasrison operator：其運算原是兩個任意資料型態，且運算結果為Boolean值。
    1. == return true if the operands are equal。等於。
    2. != return true if the operands are not equal。不等於。
    3. === 檢查operands的值，也檢查data type，兩者皆相同才return true。
    4. !== return true if the operands are of the same type but not equal, or are of different type。
    5. >
    6. >=
    7. <
    8. <=
  3. logical operator
    A B A&&B A||B
    true true true true
    true false false true
    false true false true
    false false false false
    console.log(5 > 3 && 100 > 99); // true
    console.log(3 == 6 && 100 > 99); // false
  4. typeof operator (unary)
  5. negation operator (unary)： ！
  6. increment operator (unary)： x++, x--
  7. bitwise operator
  8. arithmetic operator： +, -, *, /, +=, -=, /=, *=, **, %
