變數(variable)與賦值(assignment)
  • 變數 (variable) 內部的值可以不斷改變
  • JS 中，等號 (=) 是 "賦值" (assignment)，意旨把等號右邊的數據放到等號左邊：
    x = 5, x = x + 1
    執行後，x 值為 6。

*. 語法糖 (syntax sugar) 支援將 x = x + 1 更改為 x += 1。這在JS中極為常見。

let x; // declare a variable

x = 10; // variable assignment

console.log(x); // 看 x 的值為多少
  • 宣告變數 (declare variable)，宣告變數前，要先確認，此變數之後是否有可能被修改。基本規則：
    1. 若變數的值會變動，則用 let 來宣告變數。
    2. 若變數的值不會變動，則用 const 來宣告變數。
    3. 請勿使用 var 宣告變數。(進階JS課程會解釋)。
  • 幾個需要特別注意的規則：
    1. 用const來宣告的變數，一定需要馬上賦予初始值(initializer)。let則不需要。若用let宣告了變數，但還沒賦值，則變數的值是undefined。
    2. 用const, let宣告過的變數，都不能重複宣告。(redeclaration is not allowed)
    3. const不能做重複賦值。(reassignment is not allowed)

數字運算
數據類型(又稱資料類型) Data Type

JS 中有 7 種"基本"數據類型。這 7 種基本數據類型被稱為 primitive data type。

  1. Number - 整數與帶小數點的數字。ex：10, -3, -3.14, 200000。
    加 + , 減 -, 乘 *, 除 /
    取餘數 remindar ： %
    次方 exponentiation ： **
    x = x + 1 同 x++ 同 x+=1
    let x = 10;
    x++
    console.log(x);
    // x為11
  2. BigInt (Big Integer) - 任意長度的整數。
  3. String - 字串符。ex："......"。
  4. Boolean - true 或 false 兩種值。
  5. null - 用來代表某個故意不存在的值。
  6. undefined - 未被賦值的變數，就是undefined。
  7. symbol - unique identifier

除上述7種外，JS第八種數據類型叫做 object，屬於 non-primitive data type。JS中的object可能是 array, object 或是 function 等等。
