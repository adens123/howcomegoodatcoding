Math Object

  1. Math Object中，常用的static propertires包含：Math.PI, Math.E。

  2. 常用的static methods包含：Math.pow(x, y), Math.random(), Math.sqrt(x), Math.abs(), Math.floor(), Math.ceil()。

    • Math.pow(x, y)

      console.log(Math.pow(3, 2)); // 3的二次方，得 9。

    • Math.random()
      return的值介於0~1，涵蓋0，不涵蓋1。 [0, 1)。0 <= Math.random < 1。

      console.log(Math.random()); // 隨機出現一個0~1的數值，每次出現的都會不同。

    • Math.sqrt(x) ： 開根號 (square root)。

      console.log(Math.sqrt(2)); // 2開根號 = 1.4142135623730951

    • Math.abs() ： 絕對值 (整數) (absolute value)。

      console.log(Math.abs(-2)); // -2的絕對值 = 2

    • Math.floor() ： 向下取整數。

      console.log(Math.floor(9.99999999999)); // 9

    • Math.ceil() ： 向上取整數。

      console.log(Math.ceil(9.99999999999)); // 10
      console.log(Math.ceil(9.00000000000001)); // 10
