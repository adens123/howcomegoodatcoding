Function 的時間複雜度

  • 在電腦科學中，演算法的時間複雜度 (Time complexity) 可以描述該演算法的執行時間。

  • 直觀上來說，時間複雜度就是，在參數逐漸變大的情況下，Function執行完畢所需要的時間的成長數度。

  • JavaScript陣列中的常見methods相對應的時間複雜度為：
    • push - O(1)
    • pop - O(1)
    • shift - O(n)
    • unshift - O(n)

  • 每個瀏覽器的 JavaScript 引擎內部對於陣列的實現方法略有差異。對於不同大小的陣列，可能會採用其他更好的資料結構，例如 double-linked list, bin search tree (BST) 等等，來增強表現。

Object 語法
  • 屬於物件的 function 被稱為 method。
  • 物件的屬性與相對應的值是一種 key-value pair。
  • JaveScript Object 是一種 hashtable。
  • 獲取物件屬性的方式：
    • dot notation
    • [ ]
    let Wilson = {
       // properties (key-value pair), methods
       first_name: "Wilson",
       last_name: "Ren",
       age: 26,
       is_married: true,
       spouse: "Grace",
    };

    // dot notation
    console.log(Wilson.first_name); // Wilson

    // [ ]
    console.log(Wilson["spouse"]); // Grace
    console.log(Wilson["age"]); // 26
    let Wilson = {
       // properties (key-value pair), methods
       first_name: "Wilson",
       last_name: "Ren",
       age: 26,
       is_married: true,
       spouse: "Grace",

       sayHi() {
          console.log("Wilson says hi.");
       },

       walk() {
          console.log("Wilson is walking...");
       },

       speak(words) {
          console.log(`Wilson says ${words}`);
       }

    };

    Wilson.sayHi(); // Wilson says hi.
    Wilson.walk(); // Wilson is walking...
    Wilson.speak("今天心情不錯"); // Wilson says 今天心情不錯
  • 在method中的this關鍵字指的是調用該方法的物件。若某個function沒有調用該function的物件，則this關鍵字則是指向window物件。
    let Wilson = {
       // properties (key-value pair), methods
       first_name:"Benson" ,
       last_name: "Ren",
       age: 26,
       is_married: true,
       spouse: "Grace",

       sayHi() {
          console.log(this.first_name + " says hi");
       },

       walk() {
          console.log(this.first_name + "is walking...");
       },

       speak(words) {
          console.log(this.first_name + "says " + words);
       }

    };

    Wilson.sayHi(); // Benson says hi.
    Wilson.walk(); // Benson is walking...
    Wilson.speak("今天心情不錯"); // Benson says 今天心情不錯
  • JS中的function、array 其實都是物件(Object)。Array以及Function都是special type of objects。
    let arr = [1, 2, 3, 4,5];
    console.log(typeof arr); // object

    function hello() {
       console.log("hello world");
    };

    console.log(typeof hello); // function

    若要確認是否為array，可用下方method：

    console.log(Array.isArray(arr)); // true
