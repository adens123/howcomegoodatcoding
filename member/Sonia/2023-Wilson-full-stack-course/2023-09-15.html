常見 JavaScript 函數
  1. Case Sensitive - JavaScript 中的大小寫是有差別的。
  2. 空白鍵、換行鍵等等在 JavaScript當中會全部被忽略。
    通常來說，放到伺服器上的 JavaScript 程式碼都會被做 minification (刪除空白鍵、換行鍵)。
    Minification 可少量的減少 JS 檔案大小。
  3. JS 的註解: 單行 // 、 多行則包在 ? /* */ 內部。
  4. JS 的變數名稱開頭:
    • 文字
    • underscore (_)
    • dollar sign ($)
    • [不能用數字開頭] !!
  5. JS 內鍵關鍵字 (reserved words、keywords)，例如：null, of, if, then, in, finally, for, while, break, continue, switch, try, let, const , var 等等，不能當作變數名稱。
  6. JS 使用 Unicode 字元集合，所以 String 內部可由任何 Unicode 文字組成。
  7. Semicolons (；) 可用來分隔程式語句。Semicolons的使用是 optional。
